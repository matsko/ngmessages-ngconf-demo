# ngmessages-ngconf-demo

## Installation
1. Install NodeJS and NPM
2. Clone the repo and `cd` into it.
3. Run `npm install`.

## Running the Website
1. Install first.
2. Run `./server.sh`.
3. Visit `http://localhost:8888` using your favourite browser.
4. Enjoy

## Adding Data
1. Open `db.json`.
2. Add extra data in here that matches the existing record(s).
